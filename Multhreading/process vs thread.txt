 What is a Process?
A process is an independent program in execution.
Each process has:


Its own memory space


Resources (like file handles, system variables)


One or more threads running inside it


Think of a process as a container for program execution.
üß© Example 1:
When you open Google Chrome, your computer creates a Chrome process.
Inside it, there are multiple subprocesses ‚Äî for example:


One process for the browser UI


One process per tab


One process for plugins (like video decoding)


Each of these processes runs independently.
If one tab crashes, others can still run ‚Äî because each is its own process.
üß© Example 2:
If you open MS Word and Spotify at the same time ‚Äî
you have two separate processes running.
They don‚Äôt share memory or data directly.

‚öôÔ∏è What is a Thread?
A thread is the smallest unit of execution inside a process.
All threads inside the same process:


Share the same memory and data


Run independently and concurrently


Threads are used to do multiple things at once within the same program.
üß© Example 1:
In MS Word, you might have:
One thread for typing input
One thread for auto-save
One thread for spell-check


All these threads work together and share the same document data.
üß© Example 2 (Code Example):
In Python:
import threading
import time

def print_numbers():
    for i in range(5):
        print(f"Numbers: {i}")
        time.sleep(1)

def print_letters():
    for c in 'ABCDE':
        print(f"Letters: {c}")
        time.sleep(1)

# Both threads run in the same process
t1 = threading.Thread(target=print_numbers)
t2 = threading.Thread(target=print_letters)

t1.start()
t2.start()

t1.join()
t2.join()

üß© Output (interleaved):
Numbers: 0
Letters: A
Numbers: 1
Letters: B
...

Here, both functions run concurrently within the same process using threads.
